Sobre o Sistema

FlightOnTime é uma solução desenvolvida para o Hackathon One Brasil que utiliza machine learning para prever atrasos em voos comerciais.

Endpoints Disponíveis:

  • POST /predict: Realiza previsão de atraso
  • GET /predict/stats: Obtém estatísticas do dia

Fatores Analisados:

  • Horário do voo (pico vs. fora de pico)
  • Dia da semana
  • Rota (origem e destino)
  • Distância do percurso
  • Companhia aérea

Informações do Sistema

Este sistema foi desenvolvido como parte do Hackathon One Brasil, com o objetivo de demonstrar a aplicação de machine learning na previsão de atrasos em voos comerciais.

Como funciona:

O sistema coleta dados do voo (companhia, rota, data/hora, distância) e envia para uma API Python que executa um modelo de machine learning. O modelo analisa padrões históricos e retorna a probabilidade de atraso.

APIs Integradas:

  • API Spring Boot: Endpoint REST em Java para receber as requisições
  • API Python (Flask): Modelo de machine learning que processa os dados

Endpoints Disponíveis:

  • POST /predict: Realiza previsão de atraso de voo
  • GET /predict/stats: Obtém estatísticas do dia
